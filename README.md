# My Redux Async App (Vite + React + TS + Redux Toolkit)

Demo app that shows async data loading with Redux Toolkit `createAsyncThunk`.
Data source: https://jsonplaceholder.typicode.com/posts

## Tech stack
- Vite
- React + TypeScript
- Redux Toolkit
- React Redux

## Setup
```bash
npm install
npm run dev
```
